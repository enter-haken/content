# Static

## requirements

* [static][1] 
* tree ~ 2.0
* graphviz ~> 2.40.1
* python ~> 3.10

## build

```
$ make
```

will build the whole site.

You can find the result in the `./output` folder.

## preview

When you want to browse a preview of the site you can call

```
$ make serve
```

to start a web server.
You can see the result under `http://localhost:8000`.
